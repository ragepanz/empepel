# Empepel — Aplikasi Jual Beli Mobil

Empepel adalah aplikasi web sederhana untuk **jual beli mobil**, yang memungkinkan pengguna melihat daftar mobil, detail kendaraan, serta informasi terkait penjualan.

Aplikasi ini dibangun sebagai web application berbasis **PHP** dan dijalankan menggunakan **Docker** dengan **Nginx** sebagai web server.

---

## Gambaran Singkat

Aplikasi ini dirancang untuk mensimulasikan alur dasar platform jual beli mobil, meliputi:
- penampilan daftar mobil
- halaman detail mobil
- hingga ke menu pembayaran (belum konek payment gateway, hanya contoh saja)

Struktur project dibuat terpisah antara konfigurasi server dan kode aplikasi agar mudah dijalankan dan dikembangkan.

---

## Teknologi yang Digunakan

- **PHP** — logika backend aplikasi  
- **Blade Template** — rendering tampilan  
- **JavaScript** — interaksi dasar di sisi klien  
- **CSS** — styling tampilan  
- **Nginx** — web server  
- **Docker & Docker Compose** — environment aplikasi
